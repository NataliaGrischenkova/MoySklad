# MoySklad UI Automation Tests

Автоматизированный UI-проект для тестирования веб-приложения **МойСклад**  
(https://online.moysklad.ru) с использованием **Selenium WebDriver**, **TestNG** и **Allure**.

Проект построен с применением паттерна **Page Object Model (POM)** и покрывает
ключевые сценарии авторизации и работы с личными данными пользователя.

---

## Технологический стек

- **Java 21**
- **Maven**
- **Selenium WebDriver 4**
- **TestNG**
- **Allure Report**
- **Log4j2**
- **Lombok**
- **JavaFaker**

---
## Реализованные тесты

### LoginTest — тесты авторизации

- Проверка открытия страницы логина
- Успешная авторизация с валидными данными
- Отображение ошибок при:
    - пустом email
    - пустом пароле
    - пустых полях
    - неверном формате email
    - неверном пароле
    - email с пробелами
    - email с кириллицей
    - email с недопустимыми символами
- Проверка ограничения длины поля Email (max 255 символов)
- Проверка кликабельности:
    - ссылки **Регистрация**
    - ссылки **Забыли пароль**
- Проверка переключения языка интерфейса:
    - Русский → English
    - English → Русский

---

### AccountTest — тесты личных данных

- Проверка сохранения валидных персональных данных пользователя  
  (форма **«Личные данные»**)

Используется:
- модель `Account`
- фабрика `AccountFactory` для генерации корректных тестовых данных
